Descripción

Esta herramienta permite reconocer expresiones parentizadas, en este caso especificamente if, else, endif, while y endwhile.

Se utilizo como base el escáner previamente desarrollado aqui.

Reglas de Transición

 (q0, if, P0)  →  (q1, if)
 (q0, while, P0)  →  (q1, while) 
 (q1, if, -)  →  (q1, if) 
 (q1, while, -)  →  (q1, while) 
 (q1, else, if)  →  (q1, else) 
 (q1, endif, else)  →  (q1, bb) 
 (q1, endwhile, while)  →  (q1, b) 
 (q1, $, P0)  →  (qF, -)
